🔄 HOW IT WORKS

        ↓ (autosave every 1 second)
Saves to Database
        ↓ (Live Preview polls every 500ms)
React component fetches data
        ↓
Browser displays changes
        ↓
Admin sees update in preview panel

🛠️ TROUBLESHOOTING QUICK FIX
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Admin won't load?
  → Check database is running
  → Verify .env.local has correct variables
  → Clear browser cache

Live Preview not updating?
  → Check you're logged in
  → Clear cache and reload
  → Check browser console (F12)

Images not uploading?
  → Check /public/uploads directory exists
  → Verify file size < 5MB
  → Check file format (JPG, PNG, GIF)
